## BINARIO DECIMALE

```
1       0       0       1       0     =  1 * 2 + 1 * 16 = 2 + 16 = 18 base 10

2^4     2^3     2^2     2^1     2^0
16      8       4       2       1
```

## DECIMALE BINARIO

```
15 / 2 => 7 resto 1
 7 / 2 => 3 resto 1
 3 / 2 => 1 resto 1
 1 / 2 => 0 resto 1

15 base 10 = 1111 base 2
```

```
19 / 2 => 9 resto 1
 9 / 2 => 4 resto 1
 4 / 2 => 2 resto 0
 2 / 2 => 1 resto 0
 1 / 2 => 0 resto 1

19 base 10 = 10011 base 2
```

## ESADECIMALE


```
1101 1001 = D9 base 16 (esadecimale)
```

## CALCOLI SUGLI IP

```
ip:         192.168.10.4/24
ip binario: 11000000.10101000.00001010. 00000100
maschera:   11111111.11111111.11111111. 00000000 => 255.255.255.0

rete:       11000000.10101000.00001010. 00000000 => 192.168.10.0
broadcast:  11000000.10101000.00001010. 11111111 => 192.168.10.255
indirizzo che si usa per inviare un messaggi a tutti i dispositivi nella stessa rete

Uno di questi due indirizzi in genere si dà al router
gateway1:   11000000.10101000.00001010. 00000001 => 192.168.10.1
gateway2:   11000000.10101000.00001010. 11111110 => 192.168.10.254

numero host massimo = 2^(32 - 24) - 2 = 2^8 - 2 = 256 - 2 = 254 hosts
è il numero massimo di dispositivi collegabili a questa rete
la formula per il calcolo è: 2^(32 - cidr) - 2

rete: 3 ottetti
host: 1 ottetto
```
